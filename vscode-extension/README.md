# APIWatch VSCode Extension

0.3.0 Preview extension for APIWatch local API observability.

## Development

```bash
npm install --cache .npm-cache
npm.cmd run compile
npm.cmd test
```

Open this folder in VSCode and press F5 to launch an Extension Development Host.

Manual preview checks:

```text
APIWatch: Start Collector
APIWatch: Doctor
APIWatch: Open Dashboard
APIWatch: Show Integration Guide
APIWatch: Stop Collector
```
